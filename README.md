# ماشین حساب ترید کریپتو

ابزار قدرتمند برای محاسبه سود/زیان و مدیریت ریسک در معاملات کریپتو با پشتیبانی از صرافی‌های بین‌المللی و ایرانی.

## ویژگی‌ها

### پشتیبانی از صرافی‌های مختلف
برنامه با کارمزدهای واقعی این صرافی‌ها پیش‌تنظیم شده:

#### صرافی‌های بین‌المللی
- **Binance**: Maker 0.02% | Taker 0.04%
- **CoinEx**: Maker 0.16% | Taker 0.26%
- **Bybit**: Maker 0.02% | Taker 0.055%
- **OKX**: Maker 0.02% | Taker 0.05%
- **KuCoin**: Maker 0.02% | Taker 0.06%
- **Gate.io**: Maker 0.015% | Taker 0.05%
- **Bitget**: Maker 0.02% | Taker 0.06%
- **MEXC**: Maker 0.0% | Taker 0.02%

#### صرافی‌های ایرانی
- **Nobitex**: 0.35%

- **Exir**: 0.35%

#### تنظیمات دستی
- **دستی (Custom)**: برای صرافی‌های دیگر یا کارمزد خاص

### Maker و Taker
- **Maker**: سفارش لیمیت - کارمزد کمتر
- **Taker**: سفارش مارکت - اجرای سریعتر

### محاسبات پیشرفته
- **محاسبه حجم پوزیشن**: تعیین دقیق میزان سرمایه‌گذاری بر اساس درصد ریسک
- **محاسبات لوریج**: پشتیبانی از معاملات با لوریج دلخواه
- **محاسبه کارمزد دقیق**: بر اساس صرافی و نوع سفارش
- **نسبت R/R**: محاسبه نسبت ریسک به ریوارد
- **قیمت لیکوییدیشن**: محاسبه قیمت لیکوییدیشن پوزیشن

### پشتیبانی از معاملات
- پوزیشن‌های LONG و SHORT
- لوریج دلخواه
- تنظیم Stop Loss و Take Profit

### مدیریت ریسک
- تعیین درصد ریسک برای هر معامله
- هشدارهای هوشمند برای ریسک بالا
- توصیه‌های اتوماتیک بر اساس نسبت R/R

### رابط کاربری
- رابط گرافیکی ساده و فارسی
- انتخاب آسان صرافی از لیست
- انتخاب نوع سفارش (Maker/Taker)
- ذخیره اتوماتیک تنظیمات
- نمایش جامع نتایج

## راهنمای استفاده

### 1. انتخاب صرافی
- صرافی مورد نظر خود را از لیست انتخاب کنید
- نوع سفارش را انتخاب کنید (Maker یا Taker)
- کارمزد به صورت اتوماتیک تنظیم می‌شود

### 2. تنظیمات سرمایه
- **سرمایه کل**: مجموع سرمایه شما به USDT
- **درصد ریسک**: درصدی از سرمایه که حاضرید در هر معامله ریسک کنید (توصیه: 1-2%)
- **کارمزد صرافی**: بر اساس صرافی انتخابی تنظیم می‌شود

### 3. ورود اطلاعات معامله
- **قیمت ورود**: قیمتی که می‌خواهید وارد معامله شوید
- **قیمت استاپ لاس**: قیمت محدودیت ضرر
- **قیمت تیک پرافیت**: قیمت هدف سود
- **نوع معامله**: LONG (خرید) یا SHORT (فروش)
- **لوریج**: ضریب لوریج دلخواه

### 4. محاسبات
پس از ورود اطلاعات، روی دکمه "محاسبه" کلیک کنید. برنامه به شما نشان خواهد داد:

- صرافی و نوع سفارش انتخابی
- کارمزد دقیق بر اساس صرافی
- حجم پوزیشن مناسب
- تعداد کوین برای خرید
- میزان کارمزدها
- سود و زیان بالقوه
- نسبت ریسک/ریوارد
- قیمت لیکوییدیشن
- توصیه‌های معاملاتی

## فرمول‌های استفاده شده

### محاسبات پایه
```
میزان ریسک = سرمایه کل × (درصد ریسک / 100)
```

### حجم پوزیشن
```
LONG: درصد SL = ((قیمت ورود - Stop Loss) / قیمت ورود) × 100
SHORT: درصد SL = ((Stop Loss - قیمت ورود) / قیمت ورود) × 100

حجم پوزیشن = میزان ریسک / (درصد SL / 100)
```

### نسبت R/R
```
R/R = |سود بالقوه / زیان بالقوه|
```

### قیمت لیکوییدیشن
```
LONG: قیمت لیکوییدیشن = قیمت ورود × (1 - (1/لوریج) + (کارمزد/100))
SHORT: قیمت لیکوییدیشن = قیمت ورود × (1 + (1/لوریج) + (کارمزد/100))
```

## مثال کاربردی

فرض کنید با Binance:
- سرمایه کل: 1000 USDT
- ریسک: 2%
- صرافی: Binance (Taker 0.04%)
- قیمت ورود BTC: 45000 USDT
- Stop Loss: 44000 USDT
- Take Profit: 47000 USDT
- لوریج: 10x

برنامه محاسبه می‌کند:
- ریسک: 20 USDT
- حجم پوزیشن: 90 USDT
- کارمزد کل: 0.72 USDT
- سود بالقوه: 39.28 USDT (3.93%)
- زیان بالقوه: -20.72 USDT (-2.07%)
- R/R: 1.9:1 ✅

## توصیه‌های ایمنی

### انتخاب صرافی
- برای ترید فریکوئنت: Binance یا MEXC
- برای لیمیت اردر: همیشه Maker انتخاب کنید
- برای معاملات سریع: Taker

### مدیریت ریسک
- هرگز بیش از 2% سرمایه خود را در یک معامله ریسک نکنید
- همیشه از Stop Loss استفاده کنید
- کارمزدها را در محاسبات لحاظ کنید

### نسبت R/R
- حداقل نسبت 1.5:1 را رعایت کنید
- معاملات با R/R بالای 2:1 ایده‌آل هستند

### لوریج
- برای مبتدی‌ان: 5-10x
- برای حرفه‌ای‌ها: 10-20x
- از لوریج بالای 20x پرهیز کنید
